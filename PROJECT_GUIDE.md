# Tarhal Tourism Website - مرشد المشروع

## نظرة عامة على المشروع

موقع شركة ترحال للسياحة والسفر - موقع سياحي متكامل مبني بأحدث التقنيات لتقديم أفضل تجربة للمستخدمين.

## التقنيات المستخدمة

### Frontend
- **React 18** - مكتبة واجهة المستخدم الأساسية
- **TypeScript** - للحصول على كود آمن ومنظم
- **Vite** - أداة البناء السريعة
- **React Router 6** - للتنقل بين الصفحات
- **TailwindCSS 3** - للتصميم والتنسيق
- **Radix UI** - مكونات واجهة المستخدم
- **Lucide React** - مكتبة الأيقونات
- **Framer Motion** - للرسوم المتحركة

### Backend
- **Express.js** - خادم الويب
- **Node.js** - بيئة التشغيل
- **TypeScript** - للخادم أيضاً

### الأدوات والمكتبات الإضافية
- **@tanstack/react-query** - لإدارة البيانات
- **Zod** - للتحقق من صحة البيانات
- **Vitest** - للاختبارات

## هيكل المشروع

```
client/                     # واجهة المستخدم (Frontend)
├── components/            # المكونات القابلة لإعادة الاستخدام
│   ├── ui/               # مكونات واجهة المستخدم الأساسية
│   ├── Header.tsx        # رأس الموقع والتنقل
│   ├── Footer.tsx        # تذييل الموقع
│   └── Layout.tsx        # التخطيط العام للموقع
├── pages/                # صفحات الموقع
│   ├── Index.tsx         # الصفحة الرئيسية
│   ├── TravelOffices.tsx # صفحة المكاتب السياحية
│   ├── About.tsx         # صفحة من نحن
│   ├── Contact.tsx       # صفحة التواصل
│   └── NotFound.tsx      # صفحة 404
├── hooks/                # خطافات React المخصصة
├── lib/                  # أدوات ووظائف مساعدة
├── App.tsx               # تطبيق React الرئيسي
└── global.css            # الأنماط العامة

server/                   # الخادم (Backend)
├── routes/               # مسارات API
├── index.ts              # إعداد الخادم الرئيسي
└── node-build.ts         # بناء الإنتاج

shared/                   # أنواع البيانات المشتركة
└── api.ts                # واجهات API المشتركة
```

## صفحات الموقع

### 1. الصفحة الرئيسية (/)
تحتوي على:
- **الهيدر**: 5 صور متحولة للخلفية مع نصوص جذابة وأزرار تفاعلية
- **قسم الصور السياحية**: 3 صور متحركة مع نصوص تشجيعية وإحصائيات
- **قسم المميزات**: عرض مميزات الشركة بتصميم احترافي مع أيقونات متحركة
- **قسم معلومات عنا**: كروت متقلبة ثلاثية الأبعاد تفاعلية
- **قسم المكاتب السياحية**: عرض 5 مكاتب مميزة مع تقييمات وأزرار تفاعلية
- **قسم الخرائط**: منطقة مخصصة لخرائط جوجل مع أيقونات متحركة
- **قسم التواصل**: نموذج اتصال متكامل مع رسوم متحركة جميلة
- **الفوتر**: روابط سريعة ومعلومات الاتصال مع وسائل التواصل

### 2. صفحة المكاتب السياحية (/offices)
- **هيدر ديناميكي**: 5 صور متحولة مع شريط بحث تفاعلي
- **نظام فلترة متقدم**: بحث حسب القارة والتقييم ونوع الخدمة
- **عرض 50+ دولة**: كل دولة بمعلومات شاملة وصور جذابة
- **إحصائيات شاملة**: عدد الجولات والفنادق والمراجعات
- **نظام المفضلة**: إمكانية حفظ الوجهات المفضلة
- **روابط تفصيلية**: كل دولة لها صفحة منفصلة

### 3. صفحة تفاصيل الدولة (/offices/:countryId)
- **معلومات شاملة**: تفاصيل كاملة عن الدولة والعاصمة والمناخ
- **7 مدن سياحية**: لكل دولة مع وصف وصور ومعالم
- **معرض صور**: مجموعة صور متنوعة لكل مدينة
- **تبويبات م��ظمة**: نظرة عامة، المدن، الثقافة، معلومات عملية
- **معلومات الطقس**: معلومات مفصلة عن أفضل أوقات الزيارة
- **المأكولات التقليدية**: عرض للأطباق المحلية المميزة

### 4. صفحة من نحن (/about)
- **قصة الشركة**: تاريخ مفصل منذ التأسيس 2008
- **كروت القيم المتقلبة**: 6 كروت ثلاثية الأبعاد للقيم والمبادئ
- **الخط الزمني**: معالم مهمة في تاريخ الشركة
- **الإنجازات**: جوائز وتقديرات حصلت عليها الشركة
- **شهادات العملاء**: مراجعات حقيقية من عملاء سعداء
- **وسائل التواصل**: روابط لجميع منصات التواصل الاجتماعي

### 5. صفحة التواصل (/contact)
- **نماذج متقدمة**: نموذج شامل مع خيارات متعددة للتواصل
- **معلومات المكاتب**: تفاصيل 6 مكاتب رئيسية حول العالم
- **وسائل التواصل**: 7 منصات مختلفة للتواصل
- **خرائط تفاعلية**: منطقة مخصصة لخرائط جوجل المستقبلية
- **الأسئلة الشائعة**: إجابات على ا��استفسارات الأكثر تكراراً
- **رسوم متحركة**: تأثيرات بصرية جذابة للتفاعل

### 6. محرك البحث المتقدم
- **بحث ذكي**: بحث في الدول والمدن والجولات والفنادق
- **فلاتر متقدمة**: حسب النوع والتقييم والسعر والموقع
- **نتائج فورية**: عرض النتائج أثناء الكتابة
- **عمليات بحث شائعة**: اقتراحات للبحثات الشائعة
- **واجهة أنيقة**: تصميم عصري مع رسوم متحركة
